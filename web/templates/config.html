GeoIP Performance Analyzer

A self-hosted network performance analysis tool that scans VPN server endpoints, measures latency and throughput via OpenVPN speedtests, resolves geographic locations using MaxMind GeoLite2 databases, and visualises everything on an interactive map.

Built for homelabbers who want full visibility into their VPN fleet performance across countries — with automated scheduling, ntfy push notifications, and a dark-themed dashboard.
