Help & API

Overview

GeoIP Performance Analyzer performs mass latency testing against a list of domains/IPs, resolves their geographic location using GeoLite2 databases, and optionally runs VPN throughput tests via OpenVPN.

Quick Start

  1. Add target domains/IPs using the Servers List button (one per line).
  2. Configure Pings per IP, Timeout, and Workers as needed.
  3. Click Start Scan to begin latency testing.
  4. Optionally select rows and run VPN Speedtest to measure download/upload via VPN.

Header Buttons

☰ Servers List
Open the server list editor. One domain or IP per line. Duplicates and blank lines are removed on save.
↻ OVPN Configs
Upload a .zip containing OpenVPN configs (e.g. NordVPN ovpn.zip). Only UDP .ovpn files are extracted. If a download URL is configured in Config, the button downloads automatically instead.
↻ GeoLite2 DBs
Download latest GeoLite2-City and GeoLite2-Country databases from GitHub. Hover to compare local vs. latest release date.
⚙ Config
Open the configuration page for scheduled automation, OVPN download URL, and ntfy notification settings.

Scan Controls

Pings per IP
ICMP pings per target. Higher values give more accurate averages.
Timeout (ms)
Max wait time for a ping response.
Workers
Concurrent threads for scanning.
Start / Stop Scan
Begin or abort a latency scan.
Select All / VPN Speedtest
Select rows, then run VPN speedtest through each server's .ovpn config.

Results Table

Click column headers to sort. Search filters by domain, country, or city.

Latency color: green < 50ms, yellow < 150ms, red ≥ 150ms.

Hover latency/speed cells to see scan/speedtest timestamps.

GET Endpoints

GET /api/results

Returns all scan results as a JSON object keyed by domain.

curl http://HOST:5000/api/results
GET /api/scan/status

Current scan status: active, progress, error, stopping.

curl http://HOST:5000/api/scan/status
GET /api/v1/top/latency?n=5

Top N servers by lowest latency. Optional country parameter filters by country name (case-insensitive). Supports multiple countries via comma-separated values or repeated params.

curl "http://HOST:5000/api/v1/top/latency?n=3&country=Switzerland"
curl "http://HOST:5000/api/v1/top/latency?n=3&country=Switzerland,Germany,Austria"
[
  {"domain":"ch358.nordvpn.com","latency_ms":14.62,"ip":"217.138.203.219",
   "country":"Switzerland","city":"Zurich","rx_speed_mbps":251.56,"tx_speed_mbps":59.48},
  ...
]
GET /api/v1/top/download?n=5

Top N servers by highest download speed (Mbps). Optional country parameter filters by country name (case-insensitive). Supports multiple countries via comma-separated values or repeated params.

curl "http://HOST:5000/api/v1/top/download?n=3&country=Switzerland"
curl "http://HOST:5000/api/v1/top/download?n=3&country=Switzerland,Germany"
GET /api/v1/top/upload?n=5

Top N servers by highest upload speed (Mbps). Optional country parameter filters by country name (case-insensitive). Supports multiple countries via comma-separated values or repeated params.

curl "http://HOST:5000/api/v1/top/upload?n=3&country=Switzerland"
curl "http://HOST:5000/api/v1/top/upload?n=3&country=Switzerland,Germany"
GET /api/config

Returns current configuration (schedules, notifications, OVPN settings).

curl http://HOST:5000/api/config
GET /api/servers

Returns current servers list content.

curl http://HOST:5000/api/servers
GET /api/geolite/status

Local GeoLite2 DB dates vs. latest GitHub release.

curl http://HOST:5000/api/geolite/status
GET /api/ovpn/status

OVPN folder stats: file count and last update date.

curl http://HOST:5000/api/ovpn/status
GET /api/logs

Returns in-memory log buffer (last 200 entries, used by Log Console).

curl http://HOST:5000/api/logs
GET /api/logs/files

Lists available log files with sizes.

curl http://HOST:5000/api/logs/files
GET /api/logs/file/{name}?lines=500

Returns tail of a log file. Names: general, error, scan.

curl "http://HOST:5000/api/logs/file/error?lines=100"

POST Endpoints

POST /api/scan/start

Start a latency scan. Body: {"pings":1, "timeout":1000, "workers":10}

curl -X POST http://HOST:5000/api/scan/start \
  -H "Content-Type: application/json" \
  -d '{"pings":1,"timeout":1000,"workers":10}'
POST /api/scan/stop

Send stop signal to a running scan.

curl -X POST http://HOST:5000/api/scan/stop
POST /api/vpn-speedtest

Run VPN speedtest. Optionally specify domains, otherwise tests all servers. Body: {"domains":["ch358.nordvpn.com","de1234.nordvpn.com"]}

curl -X POST http://HOST:5000/api/vpn-speedtest \
  -H "Content-Type: application/json" \
  -d '{}'
POST /api/config

Save configuration. Body: full config JSON object.

curl -X POST http://HOST:5000/api/config \
  -H "Content-Type: application/json" \
  -d '{"schedule":{...},"notifications":{...},"ovpn":{...}}'
POST /api/servers

Save servers list. Body: {"servers":"domain1.com\ndomain2.com"}

curl -X POST http://HOST:5000/api/servers \
  -H "Content-Type: application/json" \
  -d '{"servers":"ch358.nordvpn.com\nde1234.nordvpn.com"}'
POST /api/geolite/update

Trigger GeoLite2 database download.

curl -X POST http://HOST:5000/api/geolite/update
POST /api/ovpn/upload

Upload a ZIP file with OVPN configs (multipart form).

curl -X POST http://HOST:5000/api/ovpn/upload \
  -F "file=@ovpn.zip"
POST /api/ovpn/download

Download OVPN configs from URL. Body: {"url":"https://..."} (optional, uses config if empty).

curl -X POST http://HOST:5000/api/ovpn/download \
  -H "Content-Type: application/json" \
  -d '{"url":"https://downloads.nordcdn.com/configs/archives/servers/ovpn.zip"}'
POST /api/logs/clear

Clear the in-memory log buffer (Log Console).

curl -X POST http://HOST:5000/api/logs/clear
